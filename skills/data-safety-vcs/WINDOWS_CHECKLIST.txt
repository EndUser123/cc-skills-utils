P:\ Root Handling Checklist

## Before VCS Operations at P:\ Root

1. Check current directory: pwd (must show P:\)
2. Verify: Is this P:\ root? (contains System Volume Information, .BIN)
3. If yes -> MUST use git (not sapling/sl)
4. Confirm: git status works without permission errors

## Forbidden
- sl status from P:\ root (scans system folders -> permission errors)
- sl add from P:\ root (aborts on .BIN)
- sl commit from P:\ root (fails on system folders)

## Required
- git status (always works at P:\ root)
- git add (always works)
- git commit (always works)
- git push (for remote operations)
